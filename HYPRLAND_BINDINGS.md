# Hyprland Window & Monitor Bindings Cheat Sheet

## Window Management

| Binding            | Action                      | ZMK Access                |
|--------------------|-----------------------------|---------------------------|
| SUPER W            | Close window                | Hold B/N + W              |
| SUPER J            | Toggle split                | Hold B/N + J              |
| SUPER P            | Pseudo window               | Hold B/N + P              |
| SUPER T            | Toggle floating             | Hold B/N + T              |
| SUPER F            | Fullscreen                  | Hold B/N + F              |
| SUPER CTRL F       | Tiled fullscreen            | Hold B/N + V + F          |
| SUPER ALT F        | Full width                  | Hold B/N + C + F          |
| SUPER O            | Pop window out (float/pin)  | Hold B/N + O              |
| SUPER G            | Toggle grouping             | Hold B/N + G              |
| SUPER ALT G        | Move out of group           | Hold B/N + C + G          |

---

## ZMK Key Reference

- **Hold B**              : Activates left win layer (LGUI held)
- **Hold N**              : Activates right win layer (LGUI held)
- **V**                   : CTRL (mod-tap on default/win layers)
- **C**                   : ALT (mod-tap on default/win layers)
- **Shift**               : Middle thumb keys (both hands)
- **Inner left thumb + B** : Numbers layer (combo)
- **TAB**                 : Inner right thumb on left win layer
